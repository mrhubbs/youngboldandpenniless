# Installation

Intall homebrew.

Then, install ruby via rbenv:

```sh
brew install rbenv ruby-build
rbenv init
rbenv install 3.0.0
rbenv global 3.0.0
ruby -v
```

Close and re-open terminal:

```sh
sudo gem install --user-install bundler jekyll nuggets
```

# Building

```sh
cd this/folder
jekyll build
```
